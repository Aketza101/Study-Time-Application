# Study-Time-Application
Aplicación de cronómetro diseñada para medir el tiempo de forma rápida y sencilla. Permite iniciar, pausar y reanudar sesiones con precisión. El tiempo se guarda automáticamente al salir de la pantalla del cronómetro y se recupera al volver a entrar, permitiendo continuar exactamente donde se dejó la sesión anterior sin perder el progreso.
